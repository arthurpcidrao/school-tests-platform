# Especificação Frontend Web - Next.js 14

## Pasta frontend:
- school-tests-platform/frontend

## Estrutura de Páginas
1. **Landing Page:** Apresentação do SaaS e conversão.
2. **Auth:** Login/Cadastro (Exclusivo Professor).
3. **Dashboard Principal:**
   - Lista de simulados ativos/encerrados.
   - Cards com KPIs: Média da turma, taxa de participação.
4. **Editor de Simulados:**
   - Interface para buscar questões no banco global.
   - Configuração de cronômetro e datas.
   - Seletor de alunos (upload de lista).
5. **Relatórios Detalhados:**
   - Gráficos (Chart.js/Recharts) mostrando lacunas de aprendizado por disciplina.
